Verificar problemas de conexão

Soluções para problemas de conexão com o site.

  • Problemas de conexão podem ser causados por dados armazenados em cache ou cookies corrompidos no navegador. Saiba mais sobre como realizar a limpeza clicando no tutorial.

    • Dispositivos web desktop

  • 1. Certifique-se de que o cabo do modem esteja conectado ao seu computador.
  • 2. Reinicie o modem e o roteador. Para isso, desligue ambos na tomada, espere alguns segundos e ligue-os novamente.
  • 3. Às vezes, reiniciar o computador pode resolver problemas de conexão.
  • 4. Observe a parte inferior da tela do seu computador. Se houver um ícone de rede com um "X" vermelho ou um triângulo de aviso, isso pode indicar um problema de conexão.
  • Observação: se os problemas persistirem após tentar essas etapas, entre em contato com o provedor de serviços de Internet para relatar o problema.

    • Dispositivos móveis

  • 1. Certifique-se de que o Wi-Fi ou os dados móveis estejam ativados e conectados a uma rede.
  • 2. Tente desligar e ligar o Wi-Fi ou os dados móveis para reestabelecer a conexão.
  • 3. Confira se há um indicador de dados, como "2G", "3G", "4G" ou "H", ao lado das barras de intensidade do sinal na parte de cima da tela.
  • 4. Se o indicador de dados não aparecer, talvez você esteja em uma área sem cobertura. Se possível, vá para uma área diferente ou até a rua e confira o sinal novamente.
  • Observação: se isso acontecer com frequência, entre em contato com sua provedora de serviços móveis.

    • Outras orientações gerais

  • 1. Às vezes, uma simples reinicialização do dispositivo pode resolver problemas de conexão. Desligue o dispositivo e ligue-o novamente.
  • 2. Verifique se o "Modo Avião" não está ativado, pois ele desativa todas as funções de rede sem fio.
  • 3. Mantenha o sistema operacional do dispositivo e os aplicativos atualizados, pois as atualizações podem resolver problemas de conectividade.
